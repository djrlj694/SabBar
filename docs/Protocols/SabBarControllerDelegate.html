SabBarControllerDelegate

@objc public protocol SabBarControllerDelegate

Implement this protocol to control sidebar appearance based on the current trait collection.

  • shouldShowSidebar(_:traitCollection:)

    Whenever the traitCollection is changed the SabBarController asks its sabBarDelegate (if set) to specify the visibility of the sidebar. If the sidebar is visible, the tabBar will be hidden and vice versa.

    Default values are:

    if traitCollection.horizontalSizeClass == .Regular ||
   traitCollection.verticalSizeClass == .Compact {
   return true
} else {
   return false
}
